# SIC Product Docs

This folder contains source-controlled product documentation for Sports Intelligence Cloud.

SIC is the platform. Club Vivo is the current coach-facing product and app surface.

## What Belongs Here

- Product overviews and product-scope docs.
- Current product behavior for Club Vivo.
- Pilot-specific product notes under a pilot folder.
- Future or parked product ideas under a clearly labeled future folder.

## What Should Not Go Here

- Runtime source code.
- API contracts; use `docs/api/`.
- Architecture decisions; use `docs/adr/`.
- Current system architecture maps; use `docs/architecture/`.
- Week-by-week implementation history; use `docs/progress/`.

## Current Product Area

- `docs/product/club-vivo/`
  - Club Vivo product docs for the shared coach-facing workspace.
