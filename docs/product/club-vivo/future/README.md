# Club Vivo Future And Parked Product Docs

This folder contains future, parked, or exploratory product docs for Club Vivo.

These docs preserve useful ideas, but they do not represent shipped runtime behavior unless a later source-of-truth doc explicitly promotes them.

## What Belongs Here

- Parking-lot product ideas.
- Future product planning.
- Scope notes for ideas that are not currently active runtime behavior.
- Roadmap material that should not be confused with the current product surface.

## What Should Not Go Here

- Current shipped behavior unless it is clearly labeled as context.
- Runtime source code.
- API contracts.
- Architecture decisions.
- Claims that methodology upload/source-mode, broad RAG/vector infrastructure, or a broader image-assisted intake restart is implemented.
