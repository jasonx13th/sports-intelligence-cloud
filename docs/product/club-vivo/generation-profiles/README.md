# Club Vivo Generation Profiles

This folder contains source-controlled product docs for Club Vivo generation behavior and coaching flavors.

Generation profiles describe how the product should shape generated sessions for a sport, format, methodology style, or coaching context.

## What Belongs Here

- Soccer and future sport-specific generation behavior.
- Product rules for terminology, constraints, session output, and diagram expectations.
- Notes that help the shared Session Builder and Quick Session behavior stay coherent.

## What Should Not Go Here

- Plugin packages.
- Separate apps.
- Runtime source code.
- Tenant-specific pilot rules.
- Claims that a generation profile is implemented unless the current source and architecture docs support it.
