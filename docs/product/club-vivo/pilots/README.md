# Club Vivo Pilot Docs

This folder contains pilot and test-club product notes for Club Vivo.

Pilot docs preserve evidence, examples, and club-specific direction without turning one pilot into the generic product model.

## What Belongs Here

- Pilot-specific setup and readiness notes.
- Club-specific program or methodology examples.
- Evidence from walkthroughs or trial usage that should inform future product work.

## What Should Not Go Here

- Generic Club Vivo product rules.
- Runtime source code.
- Architecture contracts.
- Source-of-truth claims that apply to every club unless they have been promoted to general product docs.
