# KSC Pilot Docs

This folder contains KSC-specific pilot and test-club product notes.

KSC is the first pilot/test club. It is not the product, and it should not define the generic Club Vivo model.

## What Belongs Here

- KSC program-type and methodology notes.
- KSC pilot examples and readiness evidence.
- KSC-specific terminology when it is clearly labeled as pilot context.

## What Should Not Go Here

- General Club Vivo product behavior.
- Product-wide rules for all clubs, academies, or organizations.
- Runtime code.
- Architecture or API contract changes.
