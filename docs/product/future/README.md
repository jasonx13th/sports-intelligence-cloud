## Concepts

- `ruta-viva.md`
  - Future cycling and active mobility analytics pillar.
- `athlete-evolution-ai.md`

## Rules

- Keep concepts clearly labeled as future.
- Do not add runtime setup instructions here unless implementation has started in an active app or service folder.
- Move a concept out of this folder only after an explicit product and architecture decision.
